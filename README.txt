== Description ==

= Pledg Payment Gateway for WooCommerce =

This plugin makes your website ready to use Pledg payment gateway to accept payments on your woocommerce store in safe way using Pledg's innovative solution.

Pledg gives access to any e-merchant the ability to authorize its clients to :
* Pay in 2 to 12 installments 
* Pay later
* Divide payment equally between multiple clients

= Main features =

Main features of this plugin:
* Easy to install and to update
* Ability to add up to 6 different Pledg payment gateways
* Fully customizable payment methods
* No SSL required 
* Complete logging of payment process (success and errors)
* The payment is processed on Pledg secured servers to prevent from any server downtime of your website

== Installation ==

1. Upload the `woocommerce-pledg` folder to the `/wp-content/plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
